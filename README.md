# AI-Powered Workplace Productivity Assistant

An AI assistant that helps automate everyday workplace tasks like writing emails, planning tasks, and answering questions.

---

## Features
- **Email Generator** – Draft professional emails by tone and audience
- **Task Planner** – Build structured daily or weekly plans
- **AI Chatbot** – Ask questions and get instant workplace assistance

## Tools Used
- ChatGPT
- Lovable.ai
- GitHub

## How to Use
1. Open `app/index.html` in your browser
2. Pick a feature
3. Enter your details and click **Generate**
4. Review and use the output
