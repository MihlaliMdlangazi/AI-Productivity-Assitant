# Task Planner Prompt

## Prompt Used:
You are a productivity coach. Based on these tasks: [user tasks], create a 
structured daily plan prioritised by urgency and importance. Include time slots 
and short breaks.

## Example:
You are a productivity coach. Based on these tasks: reply to emails, prepare 
presentation, team meeting at 2pm, review report — create a structured daily plan 
prioritised by urgency and importance.
